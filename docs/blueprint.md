# **App Name**: Sonic Canvas

## Core Features:

- Homepage Modernization: Rebuild the existing HTML/CSS landing page using React and Tailwind CSS for a modern, responsive design.
- Releases Section: Implement a dynamic and visually appealing releases section to showcase music, including the 'Bloodmoon EP' and a catalog slider with other releases.
- Projects Showcase: Develop an interactive projects section with expandable cards to highlight various projects, such as 'NORAH' and 'HABITAR P2P'.
- Cosmic Vicar Section: Design a section dedicated to Cosmic Vicar Records, detailing its services and including a contact link.
- Sound Diary Subscription: Implement a 'Sound Diary' subscription form using Netlify Forms for collecting user names and emails, with a success feedback.
- Bunker Signal Generation: Use an LLM tool to generate plausible entries for the 'Signal from the Bunker' section, each with a date, time, status, and text. The AI LLM tool will choose some from a prepared list, and decide what to write.

## Style Guidelines:

- Primary color: Deep purple (#6C4AB6) for a cosmic, modern feel.
- Background color: Dark gray (#1E1E1E) to provide contrast and a sleek aesthetic.
- Accent color: Electric green (#90EE90) for interactive elements and highlights.
- Body and headline font: 'Space Grotesk' for a computerized, techy, scientific feel, and 'Inter' for longer texts.
- Lucide Icons to provide a consistent and modern look throughout the application.
- Responsive layout using Tailwind CSS grid and flexbox to ensure optimal viewing on all devices.
- Subtle animations on hover and scroll to enhance user engagement, such as fade-ins, slides, and pulses.
